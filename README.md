# Finance-Tracker

## Personal Finance Tracker Full Stack Application

<p>This application is designed to track finance. it solves some command problems.</p>
<ul>
<li>How much money i received this month?</li>
<li>What are my big expenses like hostel rent?</li>
<li>how much money i can spend daily?</li>
<li>How much money i can save in a day or month?</li>
<li>What are coming events that need to spend some money?</li>
</ul>


<p>It is also contain several feature.</p>
<ul>
<li>Transaction History</li>
<li>Available Balance</li>
<li>Events</li>
<li>Saving</li>
<li>Low Balance Alert</li>
</ul>
